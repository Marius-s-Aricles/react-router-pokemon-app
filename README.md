# React Router Tutorial (Pokemon App)
## Installation

* Clone this repo:

```bash
git clone https://github.com/Marius-s-Aricles/react-router-pokemon-app.git
```

* Change directory to the project

```bash
cd twitter-clone
```

* Install dependencies

```bash
npm install or yarn install
```

* launch project`

```bash
npm start
```
